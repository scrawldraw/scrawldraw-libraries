# 🎨 Scrawldraw Libraries Contributor Guide

Welcome to the official **Scrawldraw Libraries** documentation! This guide helps developers, designers, and creators explore existing asset libraries, build custom drawing components, and successfully publish them to [libraries.scrawldraw.com](https://libraries.scrawldraw.com).

---

## 🚀 How to Create & Publish Your Library

### 1. Sketch Your Objects
* Open your canvas on [Scrawldraw](https://scrawldraw.com) and draw the assets or UI elements you want to package.

### 2. Group for Seamless Reuse (Recommended)
* To make sure your components act as a single unit when dragged onto a canvas, select all the individual paths/shapes of an object and press `Ctrl + G` (or click **"Group Selection"**).

### 3. Add to the Scrawldraw Library Menu
1. **Select** the grouped object on your canvas.
2. **Open the Library Panel** by clicking the "Library" icon or pressing the shortcut key `9`.
3. The selected item will appear in a square preview slot. **Click on it** (a `+` icon will appear on hover) to save it locally.
4. Repeat this process for all items in your collection.

### 4. Select and Publish
1. Inside the library panel, hover over the items you wish to export and click the **checkbox** to select them.
2. Click the blue **Publish** button.
3. Fill out the asset details (Title, Category, Description, and Author info) and submit for review!

> 🧪 **Testing Tip:** Want to double-check your `.scrawldraw` library file before sharing? Use the **"Load"** button in your library panel to re-import and test how the components scale and behave.

## 📋 Submission & Quality Guidelines

To ensure the Scrawldraw Library Marketplace remains clean, useful, and high-quality for all users, please review these rules before submitting:

| Rule | Rule Type | Requirement Details |
| :---: | :--- | :--- |
| 🌍 | **Language** | All asset names, labels, titles, and descriptions must be in **English** only. |
| 📦 | **Bundle Size** | A library must contain **at least 3 related items** grouped under a single, clear category. |
| 🛠️ | **Complexity** | Do **not** submit ultra-basic shapes that are easy to draw instantly (e.g., a simple square, line, or single arrow). |
| 🤝 | **Originality** | Do **not** duplicate or re-upload components from other existing libraries without making major changes. |
| 🎯 | **Utility** | Focus on components that offer value to the community, avoiding ultra-specific assets only relevant to your personal use case. |
| 🧱 | **Grouping** | Multi-element designs **must be grouped** so they load onto the canvas as a single unified component. |

---

## ⚠️ The Review Process

Once you hit submit, the Scrawldraw team will check your library layout:
* If elements are ungrouped, hard to use, or breaking the canvas responsiveness, we might ask for quick updates.
* High-quality submissions will be featured on our landing page showcase!

*Happy sketching, and thank you for making the Scrawldraw community more creative!*
